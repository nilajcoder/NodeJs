# 🛠️ Mongoose CRUD API

This is a simple Node.js + Express + MongoDB API that performs full CRUD operations on products using Mongoose.

---

## 🚀 Features

- Create a product
- Read all products or a single product by ID
- Update a product by ID
- Delete a product by ID

---

## 📦 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
